# 📊 Retail Sales Insights

End-to-end sales revenue analysis using Python, SQL, and Power BI.

## 🔍 Project Overview
Analyzed 1000+ sales transactions to identify revenue trends, 
top performing products, regional performance, and order patterns.

## 📈 Key Business Insights
- Identified top revenue generating products
- Analyzed monthly revenue trends across 12 months
- Compared regional sales performance (North/South/East/West)
- Calculated revenue lost from cancelled orders

## 🛠️ Tech Stack
- **Python** — Data generation and cleaning
- **Pandas** — Data manipulation
- **SQLite** — SQL queries and analysis
- **Matplotlib** — Data visualization
- **Power BI** — Interactive dashboard
